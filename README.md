# Sistema-Bancario-Em-C

Atividade realizada no início da faculdade de Engenharia De Software, na diciplina Linguagens e Técnicas de Programação.

O projeto consiste de um sistema bancário que permite:

- Cadastro de clientes em 4 categorias diferentes;
- Listagem de todos os clientes cadastrados;
- Listagem de clientes filtrados por categoria.
